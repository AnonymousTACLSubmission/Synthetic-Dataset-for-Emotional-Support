The Synthetic Dataset for emotional support has been constructed using the latest available version of Open AI GPT at the time, GPT-4, 
it is structured as multi-turn conversations between an emotional support seeker and an emotional support provider mirroring the structure 
and size of ESConv, a crowdworker-produced corpus.

This repo provides the dataset, the annotation labels for the conversations as well as the code used for generation and annotation 
of the data.
